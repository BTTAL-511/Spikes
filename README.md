  <h1>Spikes (الأشواك)</h1>

  <p>
    ملف <strong>Spikes</strong> مسؤول عن عنصر الأشواك داخل اللعبة.
    تُستخدم الأشواك كعائق خطر يسبب ضررًا للاعب عند ملامسته.
  </p>

  <h2>الوظيفة</h2>
  <ul>
    <li>إلحاق ضرر باللاعب عند الاصطدام</li>
    <li>إنهاء اللعبة أو تقليل الصحة</li>
    <li>زيادة التحدي داخل المراحل</li>
  </ul>

  <h2>طريقة العمل</h2>
  <p>
    عند دخول اللاعب في منطقة التصادم الخاصة بالأشواك،
    يتم استدعاء حدث يؤدي إلى خسارة اللاعب أو موته.
  </p>

  <h2>ملاحظات</h2>
  <p>
    يجب التأكد من تفعيل <em>Collision</em> وربط الأشواك مع كود اللاعب
    لضمان عملها بشكل صحيح.
  </p>
